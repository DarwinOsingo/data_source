What would Kenya's Cabinet decide
if only the evidence mattered?

I'm building a multi-agent LLM system that simulates Kenya's Cabinet as seven AI agents — each representing a Cabinet Secretary — debating real policy questions using only verified government documents. Personal interests and political considerations are removed by design, leaving only the data.

To make each agent credible, I'm fine-tuning a base model on Kenya-specific policy data: Budget Statements, Finance Acts, Economic Surveys, and more. To generate that training data, I need people to paste a prompt into Claude, download the resulting file, and send it back. Each prompt takes about two minutes and produces 15 high-quality training examples grounded in a real Kenyan government document.
